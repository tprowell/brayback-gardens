# Brayback Gardens — Roadmap

## Now (v1 — Build before season starts)

Core functionality to plan and track the 2026 growing season.

- [ ] **Garden spaces & simple layout map** — top-down grid for each garden, place beds/trees/structures, assign plants to locations, mark sun/shade zones
- [ ] **Plant library** — curated for Zone 9a/9b with planting windows, days to maturity, spacing, sun/water needs. Pre-seeded with their crops.
- [ ] **Planting log** — record what/where/when/how many, auto-calculate harvest dates, track status from seed to done
- [ ] **Harvest tracker** — calendar view of expected harvest windows, log actual harvests, expected vs. actual comparison
- [ ] **Task & reminder system** — auto-generated tasks from planting log, manual tasks, weekly view, assignable to either user
- [ ] **Garden notes** — quick free-text + photo capture, tagged to garden/bed/plant, timestamped, searchable
- [ ] **Dashboard** — what needs attention this week, upcoming harvests, recent notes, quick-add actions
- [ ] **Responsive web app** — mobile-first for in-garden use, full desktop experience for planning
- [ ] **Preserve planning (basic)** — flag crops tied to preserving goals, note how much to grow for target batches

## Next (v2 — During or after first season)

Improvements informed by actually using v1 through a full season.

- [ ] **Proactive planting suggestions** — "it's June, you still have time to plant X" based on zone and remaining season
- [ ] **Succession planting guide** — stagger plantings for continuous harvest (lettuce, beans, etc.)
- [ ] **Visual garden designer** — more sophisticated layout tool with drag-and-drop, measurements, plant spacing visualization
- [ ] **Companion planting engine** — suggest good neighbors, warn about bad pairings when placing plants
- [ ] **Fruit tree & perennial management** — pruning schedules, dormant spray reminders, chill hour tracking, multi-year growth tracking
- [ ] **Year-over-year analytics** — compare seasons, see what improved, identify patterns
- [ ] **Season review tool** — structured end-of-season recap: what worked, what didn't, what to change next year
- [ ] **Irrigation planning** — map water zones for the new garden, plan drip layout, track watering schedules
- [ ] **Photo timeline** — visual history of each bed/plant over the season

## Later (v3+ — Long-term vision)

Bigger features that make the tool smarter and more connected.

- [ ] **Pest & disease tracking** — log issues, get organic treatment suggestions, track what worked
- [ ] **Budget & expense tracking** — track spending on seeds, soil, materials, infrastructure
- [ ] **Preserve batch calculator** — "to make 12 jars of bread & butter pickles, grow X cucumbers" tied to recipes
- [ ] **Weather integration** — frost alerts, local forecast in dashboard, rain tracking
- [ ] **Automated irrigation integration** — connect to smart irrigation controllers, adjust watering based on weather/soil
- [ ] **AI planting advisor** — personalized recommendations based on history, preferences, and goals
- [ ] **Seed inventory management** — track seed packets, viability dates, where to order
- [ ] **Community / sharing** — share garden plans or harvest data with neighbors, local gardening groups
- [ ] **Garden journal export** — printable season summary / coffee table book style output
